# 홈페이지 구조 및 흐름

## 현재 구조 (개선 후)

```
1. 헤더
   - 좌: (빈 공간)
   - 중: [멘티명]님
   - 우: 채팅 아이콘 | 프로필 아이콘

2. 오늘 한줄 (서비스 정체성)
   - "날짜를 선택하고, 할 일을 완료해보세요"

3. 날짜 선택 (달력)
   - 주간/월간 전환
   - 블레이버스 컬러로 선택된 날짜 강조

4. 오늘의 학습
   - [목록 | 타임] 토글
   - 과제 목록 + 할 일 목록 (또는 타임테이블)
```

## 흐름: 요약 → 주요 기능 → 행동 유도

1. **요약**: 헤더 + 한줄로 "이건 학습 관리 서비스"
2. **주요 기능**: 달력(날짜 선택) + 목록/타임 toggle
3. **행동 유도**: 할 일 추가, 완료 체크, 과제 클릭

## 멘티 기준 자연스러운 순서

1. 들어오면 → "오늘 뭐 해야 하지?" (날짜 확인)
2. 달력으로 날짜 선택
3. 해당 날짜의 과제/할 일 확인
4. 완료 체크 or 과제 상세 이동
